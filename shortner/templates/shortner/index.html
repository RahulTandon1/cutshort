What is Cutshort?

Cutshort.in is a URL Shortner.

If you have very long URLs you can make them shorter and cleaner.

For example www.amazon.in/btgalitz-Rick-Roll/dp/B07K6QP8RF/ can be changed to simply cutshort.in/ricky
